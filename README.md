# Sifat's Sales Portfolio

High-performance sales platform showcasing interactive tools for outbound specialists, appointment setters, and revenue builders. Built with modern web technologies and designed for seamless user experience.

**[View Live Portfolio →](https://sifat-there.vercel.app/)**

---

## Overview

A professional portfolio platform featuring:

- **Sales Playbook** — Comprehensive objection library with proven closing techniques
- **KPI Dashboard** — Real-time performance metrics and analytics visualization
- **Call Simulator** — Interactive training tool for call scenarios and objection handling
- **Career Timeline** — Visual representation of professional progression and milestones
- **Performance Analytics** — Detailed breakdown of call metrics and conversion rates

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Access at `http://localhost:5173`

### Production Build

```bash
npm run build
```

Output generated in `dist/` directory
